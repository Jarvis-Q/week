二. 项目成员

三. demo目录说明

    assets

        base -- 目前存放的JS库文件

        mods -- 功能模块的assets

    html
        
        comments.php -- 周报评论；

        member_list.php -- 组员列表；

        nav.php -- 顶部导航；

        show.php --  周报展示

    index.php -- 首页，包含顶部导航，组员列表，周报展示及周报评论；

    login.php -- 登录页；

    write.php -- 写周报的页面；
